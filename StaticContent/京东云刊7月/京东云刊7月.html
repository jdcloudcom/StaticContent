JCS for Kubernetes Service Open Beta!

JCS for Kubernetes Service Open Beta Officially! The product is fully compatible with the standard Kubernetes API, integrating plug-ins such as JD Cloud network and storage. The service adopts the mode of fully hosted management node to provide users with easy-to-use, highly reliable and powerful container management services. The launch of the service greatly simplifies the deployment and management of Kubernetes and lowers the threshold to use it, which enhances the reliability of your application and improves the efficiency of development.

Message Queue Open Beta!

JD Cloud Message Queue is a distributed message queue service independently developed by JD Cloud. The product provides a series of highly reliable, highly available ajd high-performance message cloud services such as message publishing subscription, message query and dead letter queue. JCQ service can be used in a customer's business scenario to help reduce complexity and cost, and enable developers to focus on more important works.

Cloud Situation Awareness 2.0 Redefines Cloud SOC!

The situation awareness system is a big data security analysis product provided by JD Cloud for users. Through data modeling, behavioral learning, and intelligence correlation analysis, we can fully monitor the security, discover intrusions and attack threats, and help customers build their own security monitoring and defense systems. Cloud situation awareness can present the overall security situation of threats and exception to users through graphical and visual technologies to help customers view the security situation of resources in a more intuitive way.
